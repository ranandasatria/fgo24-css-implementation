# SNAPSOUND HERO SECTION

This project is aimed at creating the hero section for the SnapSound website. The hero section features a clean, modern design to highlight the main content of the site. It uses custom CSS for styling, providing a responsive and visually appealing layout.

## Getting Started
1. Clone this project:
```
git clone repo-url
```

2. Install the depedencies:
```
npm install
```

3. Run the project:
```
npm run dev
```

4. The project will be runnning at:
``` 
http://localhost:8080
```

## Depedencies

This project uses Node.js. Make sure you have Node.js installed on your machine.

- live-server: to simulate an HTTP server in a local environment.
